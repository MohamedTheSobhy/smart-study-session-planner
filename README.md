# Smart Study Session Planner

A productivity app built with Amazon PartyRock that helps students organize effective study sessions using generative AI.

## Features
- Personalized study plans
- Study time breakdown
- Productivity and focus tips
- Review questions for self-testing

## How It Works
Users enter:
- Subject
- Available study time
- Difficulty level
- Study goal

The app then generates a customized study session plan and related outputs.

## Built With
- Amazon PartyRock
- Generative AI
